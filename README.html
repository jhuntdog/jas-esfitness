jas-esfitness

Testground for the ES Fitness Application

Built with and powered by Nodejs and DocPad. Styled with Twitter Bootstrap and strengthened by AngularJS.

Getting Started

  1. Install DocPad

  2. Clone the project and run the server

    git clone https://github.com/jhuntdog/jas-esfitness.git
cd jas-esfitness
git submodule --init --recursive
git submodule update --recursive
npm install
docpad run

  3. Open http://localhost:9778/

  4. Start hacking away by modifying the src directory
